# Vedika
AI-powered Vedic astrology API for gaming and entertainment apps. Provides personalized astrological insights, horoscopes, birth charts (kundali), compatibility matching, numerology, and AI chatbot features. Supports 22 languages with REST API and GraphQL endpoints.
