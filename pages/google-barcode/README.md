# [Google Barcode](https://developers.google.com/vision/barcodes-overview?hl&#x3D;en)

<img src="https://raw.githubusercontent.com/apis-list/apis-list/main/apis/google-barcode/logo_256x256.png" alt="Google Barcode logo" width="200"/>

It can also detect multiple barcodes at once. It reads the following barcode formats: 1D barcodes: EAN-13, EAN-8, UPC-A. The Barcode API detects barcodes in real-time, on device, in any orientation
