# [Foreca Weather Warnings](https://developer.foreca.com/weatherdata)

<img src="https://raw.githubusercontent.com/apis-list/apis-list/main/apis/foreca-weather-warnings/logo_256x256.png" alt="Foreca Weather Warnings logo" width="200"/>

Data feed of the governmental severe weather warnings. The Foreca Warning Feed integrates weather warnings from many national warning systems worldwide and presents them in a harmonized global offering.
