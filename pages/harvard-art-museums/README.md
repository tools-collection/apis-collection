# [Harvard Art Museums](https://github.com/harvardartmuseums/api-docs)

<img src="https://raw.githubusercontent.com/apis-list/apis-list/main/apis/harvard-art-museums/logo_256x256.png" alt="Harvard Art Museums logo" width="200"/>

The Harvard Art Museums API is a REST-style service designed for developers who wish to explore and integrate the museums’ collections in their projects. The API provides direct access to the data that powers the museums&#x27; website and many other aspects of the museums.

## Details
- **Categories**: Video &amp; Media, Point of Interest, Art &amp; Design, Education
- **Type**: REST
- **Auth**: `none`
- **Free**: No
