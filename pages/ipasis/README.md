# [IPASIS](https://ipasis.com/docs)


IPASIS is a real-time bot detection and fraud prevention API that combines IP reputation, proxy/VPN detection, and email validation into a single API call. Each request returns an Interaction Trust Score (0-100) with sub-20ms response time.

## Details
- **Categories**: Security
- **Type**: REST
- **Auth**: `apiKey`
- **Free**: Yes
- **Status**: Active ✅
