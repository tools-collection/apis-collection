# [bng2latlong](https://www.getthedata.com/bng2latlong)

<img src="https://raw.githubusercontent.com/apis-list/apis-list/main/apis/bng2latlong/logo_256x256.png" alt="bng2latlong logo" width="200"/>

Simple API to convert an OSGB36 easting and northing (British National Grid) to WGS84 latitude and longitude.
