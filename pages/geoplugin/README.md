# [geoPlugin](https://www.geoplugin.com)


geoPlugin is a free, fast and reliable Geolocation Webservice API to locate your visitors based on their IP address

## Details
- **Categories**: Maps &amp; Geo
- **Type**: REST
- **Auth**: `none`
- **Free**: No
- **Status**: Graveyard 🪦
