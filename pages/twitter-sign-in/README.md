# [Twitter Sign-in](https://dev.twitter.com/web/sign-in)

<img src="https://raw.githubusercontent.com/apis-list/apis-list/main/apis/twitter-sign-in/logo_256x256.png" alt="Twitter Sign-in logo" width="200"/>

Works on websites, iOS, mobile and desktop applications. The browser log in flow is appropriate for websites and applications which are able to open or embed a web browser.

- **Categories**: Social, Authentication &amp; User Management
