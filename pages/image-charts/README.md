# [Image-Charts](https://www.image-charts.com)

<img src="https://raw.githubusercontent.com/apis-list/apis-list/main/apis/image-charts/logo_256x256.png" alt="Image-Charts logo" width="200"/>

A drop-in-replacement for the deprecated Google Image Charts. Easiest way to embed charts as image in emails, pdfs etc.
