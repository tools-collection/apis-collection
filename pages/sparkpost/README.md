# [SparkPost](https://developers.sparkpost.com)

<img src="https://raw.githubusercontent.com/apis-list/apis-list/main/apis/sparkpost/logo_256x256.png" alt="SparkPost logo" width="200"/>

SparkPost is a cloud based email delivery service that sends 25% of the worlds legitimate emails. Emails are delivered to the inbox quickly via a RESTful API or SMTP.
