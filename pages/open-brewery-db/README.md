# [Open Brewery DB](https://www.openbrewerydb.org)

<img src="https://raw.githubusercontent.com/apis-list/apis-list/main/apis/open-brewery-db/logo_256x256.png" alt="Open Brewery DB logo" width="200"/>

The goal of Open Brewery DB is to maintain an open-source, community-driven database on breweries, cideries, brewpubs, and bottleshops

## Details
- **Categories**: Point of Interest, Food &amp; Drinks
- **Type**: REST
- **Auth**: `none`
- **Free**: Yes 🆓
